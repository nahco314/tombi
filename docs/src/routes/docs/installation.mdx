# Installation

## CLI

```sh
curl -fsSL https://tombi-toml.github.io/tombi/install.sh | sh
```

Install a specific version:

```sh
curl -fsSL https://tombi-toml.github.io/tombi/install.sh | sh -s -- --version 0.2.24
```
